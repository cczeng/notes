# [Angular 表单](https://angular.io/guide/user-input)

> [Angular 表单中文版](https://angular.cn/guide/forms)  


在使用表单之前需要导入表单模块:  

```js
import { FormsModule } from '@angular/forms';

@NgModule({
    imports: [
        BrowserModule,
        FormsModule
    ]
})
```
