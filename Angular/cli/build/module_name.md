# Angular cli 使用模块名称进行打包



**使用Angular cli为每一个模块进行单独文件打包,并输出对应模块名称**  



参数以及含义:  

| —namedChunks=true\|false                 | 使用延迟加载模块的文件名               |
| ---------------------------------------- | -------------------------------------- |
| —outputHashing=none\|all\|media\|bundles | 定义输出文件名散列模式，为none则不添加 |
